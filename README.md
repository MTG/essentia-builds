# Docker images for portable builds of Essentia

Docker images for building portable [``manylinux2014``](https://www.python.org/dev/peps/pep-0599/) python wheels and executables for Essentia.

https://hub.docker.com/r/mtgupf/essentia-builds/

The images provided in this repository are based on the [official manylinux2014 images](https://github.com/pypa/manylinux) and include all pre-requisites and dependencies for building Essentia.

See https://github.com/pypa/manylinux for more details.
